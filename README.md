# ステージ理論
## ステージ1
・100MAや200MAより下でもみあい、急落がないこと。(下値余地が少ない)
・底入れしており、もみあいが続いている
・下値余地(サポートライン)が現在値より下に明確に存在

## ブレイクアウト (ステージ1 → ステージ2 への判断基準)
・価格位置が 20MA > 50MA> 100MA > 200MA となっていること。
・大陽線をつけ、出来高が直近半年の平均の3倍以上あること。

## ステージ2
・連騰に続く連騰。-5%の調整があってもすぐに買われ押し目となる。
・20MAから離れて高値で推移していること。(3%以上の乖離)
・20MAより上で必ず推移しており、ボリンジャーバンド2σの上限との間でチャートを作っている
・出来高がブレイクアウト後も多く、ボラティリティが高い。
・直近高値や上場来高値を更新し続けている

## ステージ3
・ボラティリティは以前高いが、20MAに近づいたり割ったりすることが多くなってくる
・ダブルトップの形を形成し、直近高値を超えられない
・高値を超えられない期間が長くなってきて、出来高が減っていく

## ステージ4
・50MAを明確に割るほどの出来高と伴った急落
・20MAと50Mがデッドクロスしている
・買い戻されるが20MAや50MAのラインで頭打ちとなり、触れたらすぐに下落する

## 復活 (ステージ4 → ステージ1)
・ステージ4での急落からそれなりに期間が立ち、ボラティリティが下がっている
・下値余地(サポートライン)が現在値より下に明確に存在し、センチメンタルが回復しつつある
・100MAや200MAより下に現在値があり、出来高が少ない(陰線があっても出来高が少ない)




# トレード

概要  
- マーク・ミネルヴィニの手法は「成長株のモメンタム投資」を厳格なリスク管理で運用するもの。業績・需給が加速し、株価が上場来高値近辺で強いトレンドを示す銘柄を、短中期で保有して超過収益を狙います。  
- 核心は「Trend Template（トレンドの型）」「VCP（Volatility Contraction Pattern：ボラティリティ収縮パターン）」「SEPA（Specific Entry Point Analysis：具体的エントリーポイント分析）」の3つ。  
- O’NeilのCAN SLIMやWeinsteinのStage分析と思想が近く、上昇トレンドのリーダー銘柄に絞り込み、精密なエントリーと早い損切りで「損小利大」を徹底します。  

基本哲学  
- 強い上位トレンドへの順張り（弱い銘柄の逆張りを避ける）  
- 勝ちやすい地合い・銘柄に集中（選択と集中）  
- エントリーは「具体的な一点（ピボット）」に限定し、出来高・需給で優位性を確認  
- 損失は小さく固定し、利益は伸ばす（ナンピン禁止、増し玉は含み益のみ）  
- 市場環境（指数・セクター）に整合する運用  

空売り等は禁止でロングトレードのみ。
基本的にはマークミネルヴィニのトレード手法に基づいて ブレイクアウトを狙うトレンドフォロー投資
を軸に立ち回りたい。
業績が加速し、株価が上場来高値近辺で強いトレンドを示す銘柄を、厳格なリスク管理のもとで短中期に保有します。


Trend Template（スクリーニングの型：代表的条件）  
- 価格が50・150・200日移動平均線（MA）の上  
- 50日MA > 150日MA > 200日MA  
- 200日MAが少なくとも1カ月以上上向き  
- 52週安値からの上昇が25–30％以上、かつ52週高値までの距離が25％以内（高値圏）  
- 相対力（RS）が強い（RSレーティング70以上、RSラインが高値更新など）  
- 流動性が十分（売買代金/平均出来高の基準を満たす）  
- ファンダメンタルの加速（EPS/売上高が伸び、できれば加速）


ファンダメンタルのフィルター（成長の質）  
- EPS成長率が直近四半期・通期で25％以上、売上高も拡大  
- 成長の「加速」（YoYだけでなくQoQやガイダンス上方修正）  
- 利益率・ROEが高い、キャッシュフローが健全  
- 新製品・新ビジネス、規制追い風などのカタリスト  
- セクター全体が強く、業界リーダー銘柄を選好  
- 過度な希薄化（増資）や開示の質にも注意  

VCP（ボラティリティ収縮パターン）  
- 上昇後のベース（持ち合い）で押し幅が段階的に小さくなる（例：25％→15％→8％→5％）  
- ボラと出来高が「枯れる（Dry-up）」＝供給減少で売り圧力が弱まる  
- ベース上部でピボット（明確なレジスタンス）を形成し、出来高を伴う上抜けでトレンド再開  
- 直近の収縮が最も小さくタイトであることが理想  
- 「ワイド＆ルース（乱れて広い値動き）」は品質が低く見送り  
  
SEPA（具体的エントリーポイント分析）  
- トレンドの質：MAの並び、RS、新高値圏、インデックスとの整合  
- ベースの質：収縮の連続性、タイトさ、期間、オーバーヘッド供給の少なさ  
- 触媒（カタリスト）：決算、新製品、規制変更、セクターの資金流入など  
- エントリー形態：ピボット上抜け、早期（チート）エントリー、再テストからの再上昇  
- リスク管理：初期ストップ位置、サイズ計算、失敗時の撤退パターンの明確化  




## エントリー
エントリー手順（代表例）  
- ピボットの定義：ベース上部の明確なレジスタンス（直近高値・トレンドライン・水平線）  
- トリガー：終値または場中の上抜け＋出来高増（平均比40–50％増など）  
- 価格許容範囲：ピボットからの乖離は小さく（目安3–5％）。遅れたら追わない  
- 早期（チート）エントリー：ベース内の下降トレンドライン突破や超タイトなスキィーズで小さく試す  
- 増し玉：ブレイク後の浅い押しや再圧縮からの再上昇で、含み益が乗っているときのみ 


## リスク管理・ポジションサイズ  
- 初期ストップ：ピボット割れ・直近収縮レンジ下抜けで撤退（目安3–7％、銘柄のATRやボラで調整）  
- 1トレードあたりの口座リスクを固定（例：0.5–1.0％）。サイズ＝許容損失額 ÷ ストップ距離  
- ナンピン禁止（平均下げなし）。増やすのは強さが確認され、含み益があるときだけ  
- 同時保有の相関とエクスポージャー管理（同業種過度集中を避ける）  
- 「異常（abnormal）」な値動き・出来高が出たら即撤退（計画にないリスクは持たない）

## 利益確定
20MAと50MAと100MAと200MAの4つの移動平均線を重視しており、
ボリンジャーバンド(20日、2σ)の上限に達したら一部利益確定。

利確・保有管理  
- 初動は「強さ維持」を重視：10日線・21日線の明確な割れ＋重い出来高で部分縮小/撤退  
- トレンド成熟期は50日線（10週線）などへトレーリング。大幅拡張後の過熱では段階的に利食い  
- ブレイク直後の失速（フェイルドブレイク）は機械的に切る  
- 連続の大陽線やニュース一過性の急騰は過熱に注意し、計画的に利確  

## 損切り・撤退
下落し50日移動平均線に触れたら全て売却。などを主軸としたい

マーケット環境のフィルター  
- 主要指数が50・200日線の上、ディストリビューションデー過多でない  
- セクターローテーション・リーダー交代を観察（弱った業種の保有を減らす）  
- ベア局面・不安定な地合いでは現金比率を高め、試し玉は小さく 


実践フロー（テンプレート）  
1) スクリーニング：Trend Template＋流動性＋RS＋成長の質で候補抽出  
2) チャート精査：VCPやフラットベース、タイトさ、出来高ドライアップ、上値供給の少なさ  
3) 触媒確認：決算日程、ガイダンス、製品・規制イベント  
4) プラン作成：ピボット価格、買いゾーン、初期ストップ、予定サイズ、増し玉条件  
5) 実行と管理：計画通りに買い、リスクを固定、異常時は素早い撤退  
6) トレンド成熟・変調で利確・縮小・撤退  
  
スクリーニング条件例（目安）  
- close > sma(50) and sma(50) > sma(150) and sma(150) > sma(200)  
- sma(200) が上向き（例：sma(200) > sma(200)[20]）  
- 52週高値までの距離 < 25％、52週安値からの上昇 > 30％  
- RSレーティング ≥ 70（可能なら）またはRSラインが高値更新  
- 平均出来高基準（例：50万株以上、または売買代金一定以上）  
- EPS/売上高成長率 ≥ 25％、直近で加速の兆候  
  
よくある落とし穴  
- ベースが広く乱れ（ワイド＆ルース）・上値供給過多の状態で追随  
- 一過性イベントの出来高に惑わされる（持続性のないブレイク）  
- 市場地合いを無視（指数が弱いと勝率が著しく低下）  
- ストップが広すぎ/狭すぎでボラに合わない、口座リスク過大  
- 希望的観測による保有継続やナンピン


補足・実務上のポイント  
- 数値基準は銘柄のボラ・時期に合わせて調整（小型高成長はストップ幅や許容ゾーンが広がりやすい）  
- チャートは日足でセットアップ、週足で大局の健全性を確認（Stage 2の上昇・RSのトレンド）  
- 決算・ガイダンスが近い場合はサイズを調整、想定外のギャップリスクに備える  
- エントリー前チェックリスト（例）：地合い良好／トレンド健全／ベース品質高／触媒あり／買いゾーンとストップ明確／サイズ計算済み／複数シナリオ準備
